# Image to Sketch Converter

This project converts a normal image into a pencil sketch using Python and OpenCV.

## Features
- Convert a normal image into a pencil sketch
- Implemented using OpenCV image processing techniques
- Simple and beginner friendly computer vision project

## Technologies Used
- Python
- OpenCV
- NumPy

## How to Run

1. Clone the repository

git clone https://github.com/your-username/image-sketch-project.git

2. Install dependencies

pip install -r requirements.txt

3. Run the script

python code/sketch.py
